SAP S/4HANA Cloud SDK - Release Notes

Version 1.1.1

Coming soon.

New Functionality

  • Support for SAP Java Connector (JCo) 3.0.17 for S/4HANA on-premise connectivity.
  • Support for mvn tomee:run and mvn jetty:run in TomEE and Tomcat archetypes for Cloud Foundry.

Incompatible Changes

  • S/4HANA virtual data model for OData now uses prefix “fetch” instead of “get” for navigation properties.
    For instance, a navigation property that was previously named getItem() is now named fetchItem().

Changes

  • Add DestinationType to represent different destination types.
  • Add GenericDestination as generic base interface for all destination types.
  • Add RfcDestination to represent RFC destinations.
  • Add additional methods to DestinationAccessor for additional destination types.
  • DestinationAccessor now throws DestinationOAuthTokenException for missing OAuth tokens.
  • MockUtil now throws TestConfigurationError instead of ShouldNotHappenException for test configuration issues.
  • MockUtil now redirects audit logs to the default test log instead of ignoring them.
  • Improve Javadoc of S/4HANA virtual data model.

Resolved Issues

  • Fixed Lombok using scope compile instead of provided.
  • Fixed issue with local containers not working due to missing audit logger dependency.
